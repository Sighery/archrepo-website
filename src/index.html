Sighery's Arch Linux repository

This is an Arch Linux repository holding some packages I either forked and run, or some packages from the AUR I often use.

This is all mostly automated.

Adding the repository

Adding the keys

Each package is signed with a PGP subkey I use. The database for the repository itself is also signed with that same PGP subkey. As such, this key must be imported for Pacman to verify any database update and/or package you install from this repository has been signed by me.

pacman-key --keyserver eu.pool.sks-keyservers.net --recv-key '9454A24E1B5E963078B6E9317C02D10683ADCFB8'
pacman-key --lsign-key '9454A24E1B5E963078B6E9317C02D10683ADCFB8'

Adding my repository source

You have to add these lines to your /etc/pacman.conf file:

[sighery-archrepo]
Server = https://archrepo.sighery.com/$arch

You then run pacman -Syy to update the repo list.
